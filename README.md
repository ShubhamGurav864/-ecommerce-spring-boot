🛒 E-commerce Backend (Spring Boot)
A production-ready backend system for an e-commerce platform built using Java + Spring Boot.
This project handles authentication, product management, cart, orders, and secure payments using Stripe.

🌐 Live Demo
📄 Swagger API Docs (Interactive):
https://ecommerce-spring-boot-v3ib.onrender.com/swagger-ui/index.html

📌 Features
🔐 JWT-based Authentication & Authorization
🛍️ Product Management APIs
🛒 Cart Management
📦 Order Management
💳 Stripe Payment Integration
🗄️ PostgreSQL Database Integration
✅ RESTful API Design
🐳 Dockerized Deployment
☁️ Cloud Deployment (Render)

🛠️ Tech Stack
Backend: Java, Spring Boot
Database: PostgreSQL
Authentication: JWT (JSON Web Token)
Payment Gateway: Stripe
Build Tool: Maven
Deployment: Docker + Render
API Testing: Swagger UI
